# 常用中国色参考

在用户只给出色名、且没有品牌色规范时，可用以下默认值作为提案；开始生图前向用户明确展示采用值。

| 色名 | 默认 HEX | 视觉用途 |
|---|---|---|
| 月白 | `#D6E3E8` | 冷亮主笔、月色高光 |
| 靛青 | `#1F4E79` | 深色收笔、夜色 |
| 雪青 | `#B6A7D4` | 清冷柔和主笔 |
| 石绿 | `#2D6A5F` | 山林、苔色、深色收笔 |
| 朱砂 | `#B33A3A` | 强调、印记、暖色主笔 |
| 缃色 | `#F0C36E` | 古纸、暖光、高光 |
| 玄色 | `#151515` | 背景或极深笔色 |
| 藕荷 | `#B88FA8` | 温柔紫粉过渡 |

两种前景色必须与背景保持足够明度或色相对比。若任一前景色接近绿色，使用品红色键而非绿色色键。
