# Chinese Calligraphy Video

把中国诗词、古风短句、散文与东方哲思内容，设计成具有中国色彩美学、书法气韵、意境画面和 TTS 旁白的视频。

你不需要理解程序，也不需要先决定所有技术参数。只要把内容和想要的感觉告诉 Codex，Skill 会完成内容分析、审美设计、方案评选、GPT Image 生图、书法、配色、TTS、封面和成片质检。

本文重点说明三件事：

- 如何使用这个 Skill。
- 它如何根据内容设计，而不是套用固定模板。
- “空见｜王维《鹿柴》”与“人间十八境”两个真实案例是怎样完成的。

> “空见”是本文对《鹿柴》“空山不见人”案例的简称，不是诗名，也不是对原诗的改写。

## 这个 Skill 适合什么内容

- 五言、七言诗词。
- 词、曲、古文名句。
- 古风短句合集。
- 禅意、哲思、人生感悟。
- 需要人物意境的宋式快闪视频。
- 需要狂草、行草、榜书等书法表现的文字视频。
- 微信视频号、抖音、小红书、B 站等平台的横屏或竖屏内容。
- 系列栏目，如《字有回响》、每日一诗、十八境、二十四节气等。

## 你需要提供什么

最少只要提供正文。信息越完整，第一轮越接近你想要的结果。

### 必需信息

- 完整正文，保留标点和分句。
- 如果是诗词，提供标题和作者；不知道时可以让 Skill 先核对。
- 说明它是一首完整作品，还是多句串联的合集。

### 可选信息

- 想要狂草、行草、楷书、宋体，或让 Skill 根据内容判断。
- 喜欢或不喜欢的颜色。
- 是否需要人物、山水、佛禅意境或纯书法画面。
- 横屏 `16:9` 或竖屏 `9:16`。
- 希望做快闪、沉浸朗诵或短栏目片头。
- 男声、女声、年龄感、情绪和语速。
- 参考图片、参考视频或已经认可的书法样式。
- 视频标题、期数、封面文字和发布平台。

### 不确定也可以直接交给 Skill

如果你只提供内容，Skill 会主动判断：

- 内容的核心主题和情绪转折。
- 最合适的书体与字体层级。
- 两个中国传统颜色及各自职责。
- 画面需要人物还是留白。
- TTS 应该清冷、温润、苍劲还是克制。
- 每句应该快切、慢留还是先动后静。
- 封面应该突出标题、人物还是书法。

## 推荐使用流程

### 第一步：提交原文

先让 Skill 锁定文字，不要一开始就急着生图。

```text
使用 $chinese-calligraphy-video 分析下面内容。
先逐字核对原文，暂时不要生成图片。

标题：……
作者：……
正文：……
```

如果内容来自网络短句合集，建议加一句：

```text
这些句子不是同一首诗，不要统一标注作者；请把它们作为一个连续主题来设计。
```

### 第二步：让 Skill 做内容分析

Skill 会先回答：

- 这段内容真正表达什么。
- 主要意象有哪些。
- 情绪如何开始、转折与结束。
- 画面观看路径如何变化。
- 哪些词是视觉和声音的重点。
- 哪些文化符号应该使用，哪些应该克制。

你可以这样要求：

```text
不要先给配色。先分析主题、意象、情绪弧、光线、材质、声律和观看路径。
```

### 第三步：比较 2–3 个审美方向

每个方向都应该是一个完整方案，而不只是换颜色。

方案至少包括：

- 方向名称和一句审美命题。
- 主导风格。
- 正文书体与辅助字体。
- 两个颜色的名称、HEX 和语义角色。
- 人物、背景、构图与留白。
- 显字方式、切镜节奏和停留时间。
- TTS 声线、情绪、语速与停顿。
- 不应该出现的元素。
- 审美评分与选择理由。

推荐指令：

```text
给我 3 个差异明确的方向，每个方向都要包含书体、字体层级、中国传统双色、构图、留白、人物、光线、动态和 TTS。
请按内容契合、文化一致、书法可读、配色、构图、节奏和声音评分，并推荐最优方案。
```

### 第四步：确认一个方向

确认时不要只说“好”。把最重要的取舍说清楚，后续会更稳定。

```text
选择方向 1。
保留雪青×石绿、深墨空山和现代狂草；不要人物、鹿、亭台、印章和金色装饰。
```

如果方向基本正确，但局部不满意：

```text
保留整体方向，只调整文字可读性。狂草的气势不要变，但细笔和暗笔必须在手机上看清。
```

### 第五步：先看关键画面

长视频不建议一次生成全部镜头。先看一到三个关键场景：

- 第一镜：确定系列气质。
- 情绪转折镜：检查配色和人物角色。
- 收尾镜：检查是否真正完成情绪落点。

推荐指令：

```text
先生成第一镜、最强转折镜和最后一镜让我确认。
背景不要写字，正文使用单独的狂草母版叠加。
```

### 第六步：生成全部画面与 TTS

视觉确认后，再生成全套画面和逐句 TTS。

```text
视觉方向确认。继续生成全部镜头和逐句 TTS。
旁白使用克制的青年男声，语速略慢；每句读完后保留呼吸，不要被下一镜截断。
```

如果要复用已经认可的音色：

```text
严格复用《鹿柴》的同一个 Voice ID，不要只根据“青年男声”重新选择近似音色。
```

### 第七步：制作视频和封面

```text
制作最终视频，并同时生成可直接用于微信视频号的封面。
封面必须在缩略图尺寸下看清主标题，只保留一个简短的审美标签，不要堆制作说明。
```

### 第八步：检查成片

交付前让 Skill 检查：

- 每句文字是否正确。
- 狂草在手机尺寸下是否可读。
- 双色是否清楚且有主次。
- 人物与文字是否争抢焦点。
- TTS 是否串句、截断或音色漂移。
- 实际视频抽帧是否与设计稿一致。
- 封面标题是否在微信视频号缩略图里可辨。

推荐指令：

```text
不要只检查源图片。请从最终视频实际抽帧，检查所有正文的可读性、配色、显字时机和 TTS 对齐。
```

## Skill 如何做审美设计

### 内容决定书体

- 瞬时声响、风雨、破关、决绝：适合狂草、大草或强烈榜书。
- 清幽、思念、月色、回望：适合行草或克制狂草。
- 金石、历史、警句、誓言：适合碑意、隶意或结构稳定的书法。
- 叙事长句和人物群像：可以用狂草做主标题，用宋体承担小标签和辅助信息。

狂草不是默认的“酷”，而是内容需要速度、冲突、气势或瞬间能量时才成立。

### 内容决定双色

每次只使用两个主要颜色，并给它们明确职责。

例如：

- 雪青：人声、返照、主要识别。
- 石绿：深林、青苔、收笔回声。
- 月白：文字骨架、雪光、寂静。
- 朱砂：心念、相逢、转折。
- 纯白：坚定、过关、清晰度。
- 青绿：生机、前路、希望。

如果两个颜色平均出现、没有主次，画面往往会变成装饰而不是表达。

### 内容决定人物

有些内容需要人物，有些内容最忌讳出现人物。

- 《鹿柴》写“空山不见人”，不画人物反而更准确。
- “酒逢知己”需要人物关系和杯盏动作。
- “关关难过”需要一个正在穿过关门的人。
- “偷得浮生半日闲”需要人物放慢，而不是继续保持快闪力度。

人物设计遵循“一镜一人、一人一事、一事一念”。群像只在内容明确需要重逢、对饮或相逢时出现。

### 内容决定节奏

- 建立空间：慢一点，让观众进入画面。
- 情绪转折：显字更有力，切镜更明确。
- 高潮：可以增强笔势，但不能牺牲文字辨认。
- 看破与清欢：减速、留白、延长停留。
- 有完整 TTS 时，以真实朗诵时长决定镜头，不把声音硬塞进快闪节奏。

## 使用实例一：空见｜王维《鹿柴》

### 用户可以这样说

```text
使用 $chinese-calligraphy-video，把王维《鹿柴》做成《字有回响》第一期。
我喜欢现代狂草和雪青×石绿，但不要先套模板。
请先分析这首诗，再给出审美方向、TTS 和封面设计。

空山不见人，
但闻人语响。
返景入深林，
复照青苔上。
```

### 内容分析

这首诗写的不是“空无一物”，而是声音和微光怎样让寂静被看见：

1. “空山”建立广阔寂静。
2. “但闻人语响”让不可见的人通过声音出现，同时让山显得更空。
3. “返景入深林”从听觉转入光线。
4. “复照青苔上”从空山远景收束到湿润青苔。

因此，画面不需要鹿、人物、亭台和满屏森林。主角应该是深墨留白、一次远声和一束返照。

### 入选设计

**雪青石绿 · 寂照狂草**，设计评分 `93.5/100`。

| 项目 | 设计 |
|---|---|
| 主导审美 | 现代大草的气韵与飞白 |
| 主色 | 雪青 `#B6A7D4`，承担人声、返照和主要识别 |
| 辅色 | 石绿 `#2D6A5F`，承担深林、青苔和收笔回声 |
| 背景 | 极暗空山、远林、薄雾和低位青苔 |
| 书法 | 重起笔、长锋、飞白断裂、稀疏墨点的现代狂草 |
| 构图 | 一镜一句，中央和字间保留深墨呼吸区 |
| 动态 | 逐字显影，前两句稍有声响感，后两句减速归静 |
| TTS | 清冷温润的青年男声，克制、略慢、句间留白 |
| 禁止 | 人物、鹿、亭台、印章、落款、金色边框、科技霓虹 |

### 为什么选择狂草

第二句的“人语响”和第三句的“返景”都有突然出现的瞬时能量，狂草可以把这种声音和光的势能写出来。第四句再通过重心下沉和停留，将画面重新收回到青苔与寂静。

### 成片结果

- 形式：16:9 横屏。
- 时长：18.83 秒。
- 正文：雪青 × 石绿现代狂草。
- 声音：Giggle.pro 青年男声“浪漫诗人”。
- 语速：0.88。
- 题签：愿君全屏静赏。
- 栏目：`《字有回响》01`。

### 如果想复用这个风格

```text
沿用“空见｜《鹿柴》”的寂照狂草体系：深墨留白、雪青主笔、石绿辅笔、清冷青年男声。
但请根据新内容重新判断意象、节奏和构图，不要直接复制空山背景。
```

## 使用实例二：《字有回响》03｜人间十八境

### 用户可以这样说

```text
使用 $chinese-calligraphy-video，把下面十八句话做成一支宋式人物快闪视频。
这些不是同一首诗，而是一系列串联短句。
每个句号一种意境，角色要鲜明，融合佛、禅意和中国传统双色；正文要狂草。
先分析十八句之间的精神路径，再设计每一境。

无人扶我青云志，我自踏雪至山巅。
酒逢知己千杯少，话不投机半句多。
……
一曲清歌满樽酒，人生何处不相逢。
```

### 内容分析

十八句虽然出处和语气不同，但可以串成一个人的精神行旅：

**独行 → 入世 → 情生 → 执念 → 劫后 → 看破 → 清欢**

这条路径成为视频真正的骨架。十八个场景可以变化，但人物比例、宋画气质、纸墨材质、狂草系统、章节标记和声音必须统一。

### 入选设计

**宋画十八境 · 人间一念禅**，设计评分 `94/100`。

| 项目 | 设计 |
|---|---|
| 主导审美 | 宋画人物点景与十八境快闪 |
| 文化逻辑 | 长卷游观、计白当黑、人物点景 |
| 禅意表达 | 通过缘、执、渡、空、照和放下表达，不堆佛像 |
| 人物规则 | 每镜一个明确身份、一个动作、一个念头 |
| 正文 | 每句上下两行狂草，分别对应两个分句 |
| 辅助文字 | 宋体章节标签与境号 |
| 背景 | 绢本、松烟、水墨、低饱和矿物色 |
| 光线 | 每镜只有一个叙事光源 |
| 构图 | 人物与文字左右错位，保留至少三分之一负空间 |
| 声音 | 与《鹿柴》完全相同的青年男声 Voice ID |

### 关键可读性优化

第五境“关关难过关关过，前路漫漫亦灿烂”最初在深色背景上不够清楚。最终改为：

- 上句纯白 `#F7F8F4`，表达过关时的坚定。
- 下句明亮青绿 `#05D7B3`，表达前路与生机。
- 保留深墨背景，让两行文字形成强对比。
- 不改变狂草气势，但加强主体笔画密度。

这说明配色不能只看色名，还要看文字在实际视频和手机屏幕上的可见性。

### TTS 与节奏

十八境严格复用《鹿柴》的同一 Voice ID，而不是重新寻找“相似男声”。

- 声线：青年男声“浪漫诗人”。
- 情绪：克制、平静。
- 语速：0.88。
- 组织方式：十八句独立朗诵。
- 每句开头保留进入时间，末尾保留呼吸。
- 第十七境明显放慢。
- 第十八境延长停留，以“人生何处不相逢”完成清欢收束。

原始快闪视觉约 43 秒，无法容纳十八句完整朗诵。最终有声版根据真实 TTS 时长延长到 130.55 秒。设计选择是让声音完整，而不是把朗诵加速到失去气韵。

### 封面设计

## 如何提出修改意见

建议一次只说清一个层级。

### 修改审美方向


### 修改书法



### 修改封面

## 最终验收清单

### 内容

- [ ] 原文、标点、标题和作者已经核对。
- [ ] 短句合集没有被误标成同一首诗。
- [ ] 全片具有清晰主题和情绪路径。

### 美学

- [ ] 书体与内容相符，不是为了“酷”而使用狂草。
- [ ] 两个颜色都有明确职责和主次。
- [ ] 人物、器物、纸墨和字体属于同一审美体系。
- [ ] 禅意来自留白、状态和结构，不依赖符号堆砌。

### 可读性

- [ ] 每个字的主体骨架可以辨认。
- [ ] 暗笔、细锋和飞白在手机屏幕上仍然可见。
- [ ] 没有错字、漏字、多余汉字、印章或水印。
- [ ] 封面标题在视频号缩略图尺寸下清楚。

### 声音与节奏

- [ ] TTS 音色、情绪和语速符合内容。
- [ ] 复用音色时使用同一个精确 Voice ID。
- [ ] 每句朗诵完整，没有串句和截断。
- [ ] 情绪转折、看破与收尾具有明确节奏变化。

### 成片

- [ ] 检查过关键画面和全部镜头联系表。
- [ ] 从最终视频实际抽帧检查过文字和配色。
- [ ] 正片、封面、标题、简介和栏目期数一致。

## 使用建议

- 第一次合作先做一到三张关键画面，不要立即铺满全部镜头。
- 内容越长，越需要先建立情绪路径和章节结构。
- 参考图用于说明审美，不等于允许照抄其中的文字、印章或版权元素。
- 狂草的最高标准不是“越乱越好”，而是势、气、意和可读性同时成立。
- 中国色彩美学不是颜色名称的堆叠，而是颜色与内容、光线、材质和情绪之间的关系。
- 最终以实际视频和手机端观看结果为准，不以单张高清源图为准。
