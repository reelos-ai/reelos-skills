# Anti-AI-Slop

Use this for landing pages, dashboards, decks, generated HTML, and polish passes.

## P0 Must Fix

- Default purple-blue or blue-cyan trust gradients in heroes.
- Hardcoded Tailwind indigo / violet accents when the brief did not request them.
- Emoji used as feature icons in serious UI.
- Generic three-card feature rows with interchangeable copy.
- Rounded cards with colored left-border accents.
- Fabricated metrics, such as "10x faster" or "99.9% uptime", without a source.
- Filler copy: lorem ipsum, feature one/two/three, placeholder text.
- Decorative blobs, orbs, waves, glows, and bokeh that do not communicate state or story.

## P1 Should Fix

- Accent color used more than twice per screen.
- More than about 12 raw hex values outside tokens.
- Oversized hero type inside compact panels.
- Stock-like imagery that does not reveal the real product, place, state, or subject.
- Perfectly symmetrical sections with no authored tension or density change.
- Cards used inside cards.

## Add Soul Without Noise

Use one memorable move:

- a typography contrast
- one unexpected proportion
- one product-specific microcopy phrase
- one interaction detail tied to state
- one cultural material rule

Do not add multiple motifs to compensate for weak hierarchy.
