# Craft Routing

Use this when deciding which detailed rules to load or apply.

## Routing Table

| Task | Required craft |
|---|---|
| Any typed UI or document | typography |
| Landing page, poster, deck | typography, color, anti-AI-slop |
| Dashboard, table, tool UI | typography, color, state coverage, accessibility |
| Mobile app or app flow | typography, state coverage, motion, accessibility |
| Editorial article or deck | typography, hierarchy, color restraint |
| Form or onboarding | state coverage, form validation, accessibility |
| Motion or video-adjacent UI | motion discipline, reduced-motion behavior |
| Multilingual / CJK / RTL | script pairing, line-height, bidi awareness |

## Craft Hierarchy

1. Legibility and information hierarchy.
2. Semantic color roles and contrast.
3. Interaction states and keyboard/focus behavior.
4. Responsive stability.
5. Motion restraint.
6. Distinctive taste move.

Do not spend the distinctive move before the first four layers are working.

## Reelos Interpretation

Reelos taste should feel authored, not decorated:

- Typography carries voice.
- Spacing creates confidence.
- Color marks decisions, not excitement.
- Motion confirms state or reveals structure.
- Cultural references shape rhythm and material before motifs.
