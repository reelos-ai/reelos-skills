# Advanced Case

## User input

把这个 SaaS dashboard 改得更有西式瑞士设计品味，但不要冷冰冰。

## Expected behavior

The skill should:
- Identify this as implementation/refinement.
- Use Western / Swiss / Bauhaus as the dominant system.
- Keep product clarity first.
- Add warmth through spacing, typography, functional color, and honest surfaces rather than decoration.

## Expected output

```text
Implemented
- Applied an 8pt spacing system and stricter 12-column layout.
- Reworked typography around Inter / Space Grotesk hierarchy.
- Reduced decorative gradients and converted colors into semantic status roles.
- Added warm neutral surfaces and clearer hover/focus states.

Design rationale
The direction uses Swiss clarity as the structural base: grid, type, and hierarchy carry the interface. Warmth comes from less cognitive friction, softer neutral surfaces, and precise microinteractions rather than ornament.

Verification
- Checked responsive layout, text wrapping, color contrast, and active/focus states.
```
