# Guided Choice Case

## User input

我想做一个 AI 产品官网，但不知道应该走什么设计风格。

## Expected behavior

The skill should:
- Avoid asking a long questionnaire.
- Offer three useful directions with tradeoffs.
- Keep typography-first framing.
- Invite the user to choose or let the agent pick the safest direction.

## Expected output

```text
I can take this in three directions:

1. Swiss Product Clarity - best if the product needs trust, conversion, and clean SaaS usability.
2. Frontier Intelligence - best if the product is for AI builders, investors, research, or community energy.
3. Deep Space Signal - best if the product needs immersive mystery, telemetry, and narrative impact.

My default recommendation is Frontier Intelligence if this is an AI-native product with a technical audience: compressed display for headlines, Manrope/Inter for body and UI, mono for metadata, black/warm-white base, and one signal accent.
```
