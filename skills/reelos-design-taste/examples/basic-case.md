# Basic Case

## User input

帮我给一个茶品牌官网做新中式设计方向。

## Expected behavior

The skill should:
- Identify this as design direction.
- Choose New Chinese / lifestyle rather than generic Chinese decoration.
- Load cultural aesthetics and typography references.
- Produce palette, typography, layout, material, motion, and guardrails.

## Expected output

```text
Aesthetic direction
新中式商业与墨色人文之间取平衡：用礼序建立信任，用留白和材质让茶品牌显得安静、高级、可亲近。

Design constraints
- Palette: rice background, ink text, cinnabar primary accent, ochre divider, muted teal for secondary status.
- Typography: Noto Serif SC for display/H1-H3, Noto Sans SC for UI/body, brush font only for one tagline.
- Layout: 12-column grid with garden-like scroll reveal; one focal object per viewport.
- Spacing: generous vertical rhythm, 96-128px section spacing.
- Shape: 2-4px radius, thin borders, minimal shadows.
- Material / imagery: real tea ware, paper fiber, porcelain glaze, misted landscape details.
- Motion: 0.8-1.2s brush-like reveal using cubic-bezier(0.22, 0.61, 0.36, 1).

Taste guardrails
- Avoid dragons, lantern overload, fake ancient textures, and too many red/gold accents.
- Preserve restraint, readable hierarchy, and material quietness.
```
