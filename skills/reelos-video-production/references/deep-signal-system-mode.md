# 深空信号系统风

## 定位

“深空信号系统风”是 `slides-video` 的视觉系统，不是新的主模式。它适合 AI 系统、Agent、Skill、基础设施、研究信号、工作流和方法论产品。

它不是通用科技蓝大屏，也不是赛博朋克。核心气质是：

> 深空底色、真实证据、操作台秩序、橙色激活信号、克制但明确的系统运动。

用户显式说“深空信号系统风”时，主模式仍按交付物判断：制作视频默认 `slides-video`，需要 Web 演示时使用 `slides-web`，两者都要时使用 `slides-dual`。

## 三层信息架构

每个场景只建立一个主阅读焦点，并把信息分成三层：

1. **观点层**：一句主张或一个问题，承担第一视觉层。
2. **结构层**：路径、拓扑、轨道、矩阵和闭环解释关系。
3. **证据层**：真实官网、产品截图、数字、栏目和来源承担可信度。

不能用抽象科技纹理替代真实产品证据，也不能让截图、标题和图表成为三个并列中心。

## 视觉契约

### 色彩角色

- 背景：`#050914`，接近黑的深海军蓝。
- 次级表面：`#0D1525` / `#131E31`，只用于真实面板与结构分区。
- 主文字：`#F5F7FB`。
- 辅助文字：`#9AA7BD`。
- 主信号：`#FF7A1A`，只用于当前节点、结论、进度和 CTA。
- 结构辅助：冷蓝、青、绿只区分类别，不得与橙色争夺主焦点。

不使用大面积紫蓝渐变、玻璃卡片、霓虹描边或无语义的辉光。

### 字体角色

- Display：中文粗黑体，承担一屏唯一主结论。
- Body：中性无衬线，承担两行以内解释。
- Meta：等宽字体，仅用于编号、状态、栏目和技术标签。
- 正文不得使用等宽字体；一屏不超过三种字号重心。

### 稳定 Chrome

顶部品牌、场景编号、技术标签和底部进度线可以形成稳定 Chrome。Chrome 必须保持低对比，不参与主焦点竞争。

## 构图语法

| 信息任务 | 推荐构图 | 禁止替代 |
| --- | --- | --- |
| 多到一 | 轨道汇聚、过滤、中心锁定 | 随机飞入 |
| 研究体系 | 非对称拓扑、主干与节点 | 再做一个中心圆环 |
| 并列资产 | 矩阵、分栏、连续面板 | 卡片套卡片 |
| 学习步骤 | 阶梯、路径、页面状态切换 | 同一截图反复缩放 |
| 因果传导 | 路径、节点点亮、流向粒子 | 装饰线乱跑 |
| 长期反馈 | 闭环轨迹、反馈点运行 | 与普通流程共用同一箭头 |
| 真实产品 | 浏览器框、局部聚光、幽灵鼠标 | 模糊氛围图 |
| 核心主张 | Kinetic Typography、一次 Sheen | 持续弹跳或闪烁 |
| 行动入口 | 品牌锁定、网址打字 | 通用按钮模板 |

圆形只承担“汇聚、循环、锁定”。拓扑只承担“研究结构”。矩阵只承担“并列资产”。同一种形状可以成为品牌母题，但不能成为每一幕的默认布局。

## 自动选效

先标记句子关系，再选择候选动效：

```text
并列信息 -> 卡片矩阵或分栏
因果关系 -> 路径与节点传导
从多到一 -> 汇聚与过滤
步骤推进 -> 阶梯、进度或页面切换
长期循环 -> 闭环轨迹
观点转折 -> Kinetic Typography
真实产品 -> 截图、聚光、幽灵鼠标
最终行动 -> 品牌锁定与网址打字
```

候选平分时依次选择：

1. 静音时更容易看懂的方案。
2. 与上一幕构图差异更大的方案。
3. 更易按帧复现、渲染更稳定的方案。

## VFX 预算

每幕最多 1 个主效果和 2 个低强度辅助效果。每 2–4 秒需要一次有意义的状态变化，但不要求所有元素持续运动。

推荐强度曲线：

```text
钩子 4 -> 身份 2 -> 研究结构 3 -> 内容证据 3
-> 学习路径 3 -> 长期回路 5 -> 核心主张 4 -> CTA 3
```

全片只保留一个强度 5 的英雄时刻。其他场景不能通过增加辉光、粒子、震动或高速镜头与它竞争。

## 反模式

- 假终端、代码雨、无数据意义的 HUD 和霓虹线。
- 紫蓝渐变球、光斑、玻璃卡片堆叠和装饰性 3D。
- 连续三个场景使用中心圆环、四卡阵列或同一种淡入。
- 持续发光、持续弹跳、大幅摇镜和纯炫技转场。
- 伪造产品能力、数字、界面和交互。
- 用颜色变化冒充构图变化。
- 让氛围层压低文字或产品界面的可读性。

## 预审清单

1. 前 3 秒是否已经建立“工具很多、系统稀缺”或同等级反差？
2. 第一帧是否可直接作为封面？
3. 每个场景能否明确归类为观点、结构或证据主导？
4. 相邻场景是否改变了信息关系与构图骨架？
5. 真实产品是否清晰可核验？
6. 英雄时刻是否唯一且落在最重要的句意上？
7. 静音观看时能否理解多到一、并列、路径和闭环？

## 调用示例

```text
使用 $reelos-video-production，研究 https://reelos.ai，制作 60 秒 Slide Story 产品宣传 MP4。
视觉采用“深空信号系统风”：真实产品证据，观点层/结构层/证据层分明；
自动动效导演按语义选择汇聚、拓扑、矩阵、路径、闭环、动排或界面叙事；
相邻场景避免重复构图，全片只保留一个强度 5 的英雄时刻。
完成低清预览、静音审查、音画同步和 100 分终审后再输出成片。
```

简写触发：

```text
用深空信号系统风制作，Slide Story，真实产品证据，自动选效，VFX 终审。
```
