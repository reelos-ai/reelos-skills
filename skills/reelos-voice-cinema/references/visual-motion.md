# Visual and Motion Direction

Use this reference when choosing style, layout, and motion.

## Lanes

### Material-led

Use when the user wants low-cost video with varied visuals. External footage is the main picture. Remotion adds:

- title area
- subtitles
- keyword tags
- mild overlays
- simple transitions

### MG-led

Use when content is abstract, technical, brand-oriented, or diagram-heavy. Remotion creates:

- causal loops
- variable graphs
- system interfaces
- comparison boards
- product-film particles/core activation
- text-driven scenes

### Hybrid

Use when free footage sets the mood but diagrams are required for comprehension.

## Good Defaults

- AI/Agent/engineering: dark digital space, restrained grid, cold white text, amber/cyan accents.
- Philosophy/classic culture: natural material, paper/ink/gold/cinnabar, calm pacing.
- Brand film: black background, negative space, cold white typography, limited gold highlights.
- Viral explainer: stronger subtitle rhythm, fast keywords, tighter cuts.

## Motion Brief

Before building complex videos, write a short internal motion brief:

```text
Content task:
Audience:
Visual lane:
Main motion skeleton:
Chapters:
Main moving object per chapter:
Timing binding:
Forbidden directions:
```

For AI Agent / systems content, prefer:

- observation → state → cause → intervention → feedback
- old model → new model comparison
- variable activation
- feedback loop
- ladder/evolution timeline

## Avoid

- fake HUD clutter
- tiny thin lines
- purple/blue generic AI gradients
- random flying labels with no information role
- all-MG slideshow when the user asked for footage as background
- blurry or 3D-looking Chinese title text
