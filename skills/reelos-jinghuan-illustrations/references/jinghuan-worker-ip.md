# 镜环小工 IP

## 角色定义

镜环小工是 ReelOS 中文正文配图的固定视觉 IP。

默认每张图都要出现镜环小工。镜环小工不是吉祥物，不是贴纸，不是可爱装饰，而是正在认真参与系统运转的荒诞工作者。

角色设定参考图在 `assets/reelos/jinghuan-worker-character-sheet.png`。它只用于校准角色比例、同心镜头环脸、橙色内芯和冷静干活感，不要逐格复刻设定图里的动作。

用户不需要说出“镜环小工”这个名字。只要任务要求 ReelOS 风格、这个风格、品牌正文配图、logo 色调配图或 ReelOS 手绘插图，就默认把镜环小工放进画面，并让它承担核心动作。

## 外形

- 深蓝近黑的小圆柱、深色豆形或略不规则黑盒身体。
- 一个明显的同心镜头环脸：米白外环、黑色环、温暖橙色内芯、深色中心。
- 可以有很弱的白色圆点眼睛，但眼睛不能抢走“镜头环脸”的识别。
- 细腿，偶尔有细胳膊。
- 身体可以变成圆柱、黑盒、镜头口、旋钮、漏斗、洞口、机器内部黑块。
- 轮廓略微不规则，有手绘感。
- 表情空、呆、冷静、认真。
- 允许少量 ReelOS logo 色：深海军蓝/近黑、米白、橙色、少量蓝紫辅助线。

## 性格

- 很认真，但做的事有点荒诞。
- 像一个低调的系统操作员，也像住在 OS 里的镜头观察员。
- 冷幽默，不卖萌。
- 有点笨拙，但不蠢。
- 像在白板草图里真的负责某个工作。

## 常见职责

让镜环小工承担核心动作：

- 搬运素材。
- 拉线汇聚信息源。
- 用镜头环观察、对焦、校准内容。
- 卡在断点里。
- 在机器里操作“判断”杆。
- 变成筛选漏斗。
- 变成系统入口、旋钮、镜头口或黑盒端口。
- 切开“素材鱼”。
- 盖章承接话术。
- 牵着承接路径。
- 举警告牌看坑。
- 从洞里伸手但接不住内容。
- 在旁边搬砖、搭桥、开门、分拣、记录。

## 禁止

- 不要把镜环小工画成过度可爱的吉祥物。
- 不要画成儿童卡通角色。
- 不要给镜环小工复杂服装、表情包、闪亮眼睛或大笑表情。
- 不要让镜环小工只是站在角落里看。
- 不要让镜环小工抢走结构表达。
- 不要把镜环小工画得太商业、太圆润、太精致。
- 不要把它画成普通机器人、摄像头 App 图标或 3D 科技设备。

## 判断标准

如果去掉镜环小工，图的核心隐喻还能完全成立，说明角色太装饰了；要重写提示词，让镜环小工成为动作主体。
