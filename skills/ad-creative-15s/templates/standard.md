# 标准输出模板（Standard）

## 模板格式

```
画面：[一段话切镜脚本，包含时间点切分、镜头语言、视觉元素]

旁白：[一句台词/文案，≤22个字中文或≤15词英文]（若无旁白则写"无（纯视觉）"）

音乐：[风格 + BPM + Drop点说明]
```

## 模板使用说明

### 画面 部分要求

1. **必须包含时间点**：0-2s / 2-5s / 5-9s / 9-12s / 12-15s
2. **丰富镜头语言**：特写、推镜、快切、旋转、定格、变焦、拉远、环绕等
3. **密集视觉变化**：每2-3秒必有新元素或新动作
4. **模式区分**：
   - **Mode 1**：纯粹视觉隐喻，无任何品牌/产品信息
   - **Mode 2**：产品信息自然融入，用视觉隐喻而非硬广
5. **视觉隐喻**：不直接硬广，用比喻（像窗花贴满、像光道分流等）
6. **一句话完成**：不分段，流畅阅读

### 旁白 部分要求

1. **字数限制**：中文≤22字，英文≤15词
2. **模式区分**：
   - **Mode 1**：诗意/哲理/情感共鸣，无品牌
   - **Mode 2**：可包含品牌名或核心卖点
3. **朗朗上口**：有节奏感，易记
4. **情绪匹配**：与整体风格一致
5. **可选无旁白**：标注"无（纯视觉）"

### 音乐 部分要求

1. **明确风格**：禅意电子、国潮、House、Trap、史诗电子、Ambient、钢琴等
2. **明确BPM**：可以是范围（110-120）或具体值（128）
3. **标注Drop点**：
   - 单Drop：通常 10-13s
   - 双Drop：通常 8s 和 12s
   - 特殊说明：如"渐进式无明显Drop"
4. **额外说明**（可选）：尾段特色（铃音、鼓点、弦乐等）

## 质量检查清单

生成脚本后，请确认：

- [ ] 画面 包含所有时间点（0-2s/2-5s/5-9s/9-12s/12-15s）
- [ ] 画面 镜头语言丰富（≥3种运镜方式）
- [ ] 画面 每2-3秒有视觉变化
- [ ] 旁白 字数符合要求（中文≤22字，英文≤15词）
- [ ] 旁白 与整体风格匹配
- [ ] 音乐 有明确风格、BPM、Drop点
- [ ] 整体风格统一
- [ ] **Mode 1**：无任何品牌/产品/商业信息
- [ ] **Mode 2**：产品信息自然融入，非硬广

## 常见错误与修正

### 错误 1：时间点缺失
```
画面：特写微笑，快切灯笼，推镜春联，拉远烟花，定镜Logo。
```
**问题**：没有标注时间点

**修正**：
```
画面：0-2s特写微笑，2-5s快切灯笼，5-9s推镜春联，9-12s拉远烟花，12-15s定镜收尾。
```

### 错误 2：Mode 1 混入品牌信息
```
画面：...12-15s定镜Logo闪耀品牌弹出。
```
**问题**：纯创意模式不应有品牌信息

**修正**：
```
画面：...12-15s定镜回眸微笑光粒散去余韵悠长。
```

### 错误 3：旁白过长
```
旁白：这是一段关于旅行和成长的故事，每一步都让我们成为更好的自己。（28字）
```
**问题**：超过22字限制

**修正**：
```
旁白：每一步，都成为了更好的自己。（14字）
```

### 错误 4：音乐信息不完整
```
音乐：欢快音乐
```
**问题**：没有风格、BPM、Drop点

**修正**：
```
音乐：国潮电子115BPM，12s Drop。
```
