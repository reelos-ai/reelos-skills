# 切镜模板库

## 模板 A：节日氛围（FESTIVAL_VIBES）

### 适用场景
- 春节/新年/圣诞/情人节等节日
- 文化表达、仪式感
- 温暖氛围、节日祝福

### 视觉元素
- **节庆元素**：灯笼、窗花、春联、红包、烟花、福字、鞭炮、圣诞树、雪花
- **色彩**：红色、金色、橙色、暖黄
- **材质**：绸缎、金粉、光效、纸质
- **动效**：跳动、旋转、飘落、炸开、闪耀

### 节奏特征
- **0-2s**：快速切镜，节日元素密集出现
- **2-5s**：中速，核心元素展开
- **5-9s**：渐强，氛围渲染
- **9-12s**：高潮，情绪爆发
- **12-15s**：回归温暖，收尾

### BGM建议
- 风格：国潮电子、民族融合、喜庆节拍
- BPM：110-120
- Drop点：10秒温暖Drop，尾段铃音/鼓点

### 示例句式

**Mode 1（纯创意）**：
```
0-2s特写[人物动作]+[节日元素]一闪，
2-5s快切[节日物件]旋转飘舞氛围渲染，
5-9s推镜[对称元素]点亮[文化符号]光带环绕，
9-12s拉远[温暖场景]光晕扩散情绪共鸣，
12-15s定镜[祝福动作]+[节日意象]闪耀收尾
```

**Mode 2（品牌广告）**：
```
0-2s特写[人物动作]+[节日元素]一闪，
2-5s快切[节日物件]牵出[产品流程]像窗花贴满画面，
5-9s推镜[对称元素]点亮[产品点]横批落下[节日雨]，
9-12s拉远[烟花/大场景]炸出[品牌概念]像[节日隐喻]，
12-15s定镜[祝福动作]+Logo闪耀
```

## 模板 B：时尚视觉（FASHION_VISUAL）

### 适用场景
- 时尚潮流、街头文化
- 年轻受众
- 音乐节/派对
- 品牌形象

### 视觉元素
- **时尚元素**：T台、闪光灯、霓虹灯、灯牌、HUD线条、几何图形
- **色彩**：黑白对比、霓虹色、金属光泽
- **材质**：皮革、金属、光线、玻璃
- **动效**：卡点、闪烁、扫描、粒子流、定格

### 节奏特征
- **0-2s**：强节拍卡点，定格+闪烁
- **2-5s**：步伐节奏，元素卡点出现
- **5-9s**：推镜材质特写，信息密集
- **9-12s**：360旋转或多角度快切，高潮爆发
- **12-15s**：定镜收尾，余韵

### BGM建议
- 风格：House、Trap、Future Bass
- BPM：120-130
- Drop点：8秒和12秒双Drop

### 示例句式

**Mode 1（纯创意）**：
```
0-2s定格[姿态]+霓虹闪+HUD扫描，
2-5s快切[步伐]卡点[视觉元素]依次点亮，
5-9s推镜[材质特写]+粒子流转[主题]视觉展开，
9-12s旋转360度[场景切换]态度视觉化，
12-15s定镜[酷飒动作]+[视觉余韵]
```

**Mode 2（品牌广告）**：
```
0-2s定格[姿态]+霓虹闪+HUD扫描，
2-5s快切[步伐]卡点[产品点]像灯牌依次点亮，
5-9s推镜[材质特写]+粒子流转[信息]文字炸开，
9-12s旋转360度[品牌概念]像光道自动分流，
12-15s定镜[酷飒动作]+霓虹Logo闪耀
```

## 模板 C：史诗叙事（EPIC_NARRATIVE）

### 适用场景
- 奇幻、武侠、仙侠
- 游戏推广
- 宏大叙事、视觉震撼
- 二次元文化

### 视觉元素
- **魔幻元素**：法印、能量刃、光阵、粒子、冲击波、符文
- **色彩**：紫色、蓝色、金色、魔法光
- **材质**：能量、仙气、光效
- **动效**：慢动作、爆发、碎裂、重组、穿梭

### 节奏特征
- **0-2s**：慢动作营造氛围，法印/能量展开
- **2-5s**：快切能量爆发，文字碎裂重组
- **5-9s**：冲击变焦，阵法运转
- **9-12s**：粒子爆发，高潮冲击
- **12-15s**：落地回归，仙气/余韵

### BGM建议
- 风格：史诗电子、Cinematic、交响乐
- BPM：130-150
- Drop点：11秒重型Drop

### 示例句式

**Mode 1（纯创意）**：
```
0-2s慢动作[施法动作]+[能量]光阵展开，
2-5s快切[能量爆发][视觉隐喻]碎裂重组，
5-9s冲击变焦[核心主题]像阵法自动运转发光连线，
9-12s定格粒子爆发[高潮意象]冲破屏幕，
12-15s落地回眸+[意境]缭绕余韵
```

**Mode 2（品牌广告）**：
```
0-2s慢动作[施法动作]+法印光阵展开，
2-5s快切能量刃劈出[产品流程]文字碎裂重组，
5-9s冲击变焦[产品点]像阵法自动运转发光连线，
9-12s定格粒子爆发[品牌概念]化成千军万马冲破屏幕，
12-15s落地回眸+Logo浮现仙气缭绕
```

## 模板 D：极简艺术（MINIMAL_ART）

### 适用场景
- 哲学思考、禅意表达
- 高端品牌、科技产品
- 商务场景、艺术短片
- 极简美学

### 视觉元素
- **极简元素**：留白、几何线条、材质质感、微动效、光影
- **色彩**：黑白灰、莫兰迪色、金属色、单一强调色
- **材质**：玻璃、金属、大理石、木纹、皮革
- **动效**：平滑推进、精准卡点、微旋转、呼吸感

### 节奏特征
- **0-2s**：缓慢推镜，细节/质感
- **2-5s**：留白处文字/元素逐一显现
- **5-9s**：材质特写，微动效精准传递
- **9-12s**：空间拉远，极简轮廓
- **12-15s**：定格留白，居中发光

### BGM建议
- 风格：Ambient、Minimal Techno
- BPM：90-110
- Drop点：12秒单一精准Drop

### 示例句式

**Mode 1（纯创意）**：
```
0-2s缓推[细节特写]+几何线条浮现，
2-5s留白[主题元素]如光影依次显现，
5-9s材质特写[核心概念]微动效精准传递，
9-12s空间拉远[升华意象]极简线条勾勒轮廓，
12-15s定格留白+[意境]居中发光
```

**Mode 2（品牌广告）**：
```
0-2s缓推[细节特写]+几何线条浮现，
2-5s留白处[产品流程]文字如光影依次显现，
5-9s材质特写[产品点]微动效精准传递，
9-12s空间拉远[品牌概念]极简线条勾勒轮廓，
12-15s定格留白+Logo居中发光
```

## 模板 E：情感共鸣（EMOTIONAL_CONNECT）

### 适用场景
- 人文故事、情感表达
- 回忆杀、成长感悟
- 温情主题、治愈系
- 旅行vlog、日常记录

### 视觉元素
- **情感元素**：人物特写、光影、老照片、回忆碎片、自然景色
- **色彩**：暖黄、柔和饱和色、日落色、胶片色调
- **材质**：柔光、光晕、雾气、光粒
- **动效**：渐进、流动、飘散、定格又流动

### 节奏特征
- **0-2s**：人物/场景特写，光影渐开
- **2-5s**：回忆/故事碎片快切如翻页
- **5-9s**：慢推镜，细节展开，情绪积累
- **9-12s**：情绪高潮，色彩饱和，时光涌动
- **12-15s**：回归平静，温暖收尾

### BGM建议
- 风格：钢琴、吉他、Indie、情感电子
- BPM：100-120
- Drop点：10秒情绪爆发点

### 示例句式

**Mode 1（纯创意）**：
```
0-2s[人物特写][光影]渐开，
2-5s快切[记忆碎片]画面如相册翻页，
5-9s慢推镜[细节]回忆飘出碎片浮现，
9-12s情绪高潮色彩饱和画面定格又流动时光涌动，
12-15s[回归动作]光粒散去只余温暖
```

## 模板 F：品牌广告（BRAND_AD）（仅 Mode 2）

### 适用场景
- 产品推广、品牌宣传
- 电商促销、App下载
- 限时活动、营销事件

### 视觉元素
- **信息元素**：产品界面、功能展示、用户场景、CTA按钮
- **创意元素**：视觉隐喻、粒子转化、光道连线
- **色彩**：品牌色为主，强调色突出
- **动效**：卡牌翻转、流程连线、粒子聚散、Logo闪耀

### 节奏特征
- **0-2s**：痛点/吸引点/视觉冲击
- **2-5s**：产品功能/类型展示
- **5-9s**：核心卖点视觉化
- **9-12s**：产品价值深化
- **12-15s**：强CTA + Logo + 旁白

### BGM建议
- 风格：节奏明确的流行/电子/科技感
- BPM：110-130
- Drop点：10-13秒CTA时刻

### 示例句式

```
0-2s[视觉冲击/痛点场景]大字弹出，
2-5s快切[产品功能]像[创意隐喻]依次展开，
5-9s推镜[核心卖点]视觉化[隐喻]自动运转，
9-12s[产品价值]全屏展示粒子炸开，
12-15s定镜Logo闪耀[CTA]弹出
```

## 选择模板的决策树

```
用户意图
├── 包含"节日/春节/新年/圣诞/情人节" → 节日氛围（FESTIVAL_VIBES）
├── 包含"时尚/潮流/动感/年轻/街头" → 时尚视觉（FASHION_VISUAL）
├── 包含"游戏/仙侠/魔幻/特效/史诗" → 史诗叙事（EPIC_NARRATIVE）
├── 包含"高级/奢华/极简/商务/禅意" → 极简艺术（MINIMAL_ART）
├── 包含"回忆/情感/成长/温暖/治愈" → 情感共鸣（EMOTIONAL_CONNECT）
├── 包含"促销/优惠/下载/转化"（Mode 2） → 品牌广告（BRAND_AD）
└── 其他 → 根据图片分析选择或询问用户
```
