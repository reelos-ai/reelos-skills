| 速度 | BPM范围 | 情绪 | 适合 |

## Drop 点设计

### 单 Drop（10-13s）
- **适合**：极简、奢华、稳重、情感
- **设计**：0-10s Build-up → 10-13s Drop → 13-15s 收尾

### 双 Drop（8s+12s）
- **适合**：时尚、动感、年轻
- **设计**：0-8s 第一波 → 8s Drop → 8-12s 第二波 → 12s 大Drop → 12-15s 高潮

### 渐进式（无明显 Drop）
- **适合**：舒缓、温暖、情感
- **设计**：0-15s 逐渐加强，无突然爆发

## 音效点位建议（SFX）

### 0-5秒（开场）
- 冲击音：画面出现
- 扫过音：HUD扫描、光效
- 粒子音：元素飘落

### 5-10秒（展开）
- 打字音：文字出现
- 卡点音：节拍同步
- 转场音：画面切换

### 10-15秒（高潮收尾）
- 爆炸音：粒子爆发
- Logo音：品牌标识出现（仅 Mode 2）
- 收尾音：余韵延续

## BGM 选择原则

### 理想的 BGM
- 器乐为主（不干扰旁白）
- 节奏稳定（便于卡点）
- 情绪明确（强化视觉）
- Drop清晰（便于同步设计）
- 授权清晰（可商用）

### 避免
- 过于复杂的旋律（抢占注意力）
- 人声过多（与旁白冲突）
- 情绪不匹配（悲伤配促销）
- BPM突变（节奏不稳定）
- 版权问题（使用未授权音乐）

## 使用流程

1. **确定风格** → 查快速选择表
2. **匹配 BPM** → 参考情绪映射
3. **找准 Drop** → 选择Drop设计模式
4. **同步视觉** → 切镜节奏与音乐节拍配合
5. **添加音效** → 参考音效点位建议
6. **详细参考** → 查阅 `reference/bgm_database.md`
