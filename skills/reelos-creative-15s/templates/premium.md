## Premium 部分详细说明

### 字幕（字幕排版建议）

**包含内容**：
1. **时间点**：每个字幕出现的时间段
2. **位置**：顶部/中部/底部/左侧/右侧
3. **样式**：字体类型、颜色、特效
4. **动效**：淡入淡出/放大缩小/滑入/闪烁等
5. **内容**：具体文字

**排版原则**：
- **可读性优先**：不遮挡主体人物
- **风格匹配**：
  - 节日：金色描边、毛笔字、喜庆字体
  - 时尚：霓虹色、英文、竖排/跑马灯
  - 仙侠：古风字体、篆刻、金色
  - 极简：无衬线、单色、留白
  - 情感：手写体、柔和色、渐变
- **节奏配合**：重要信息在Drop点出现

### 音效（音效点位）

**包含内容**：
1. **时间点**：音效出现的精确秒数
2. **音效类型**：具体声音描述
3. **音效作用**：配合什么视觉元素

**音效分类**：
- **冲击类**：爆炸、冲击波、砰、咚
- **过渡类**：唰、嗖、扫描音
- **氛围类**：风声、铃铛、风铃、仙气
- **节拍类**：鼓点、咔嚓、啪
- **电子类**：HUD扫描、数据流、科技音
- **品牌类**：Logo音、收尾音（仅 Mode 2）

**添加原则**：
- 0s：开场定调
- 重拍/卡点：强化节奏
- Drop点：音效最强
- 15s：收尾音

### 备选旁白（3个备选台词）

**提供 3 个备选**，满足不同需求：
1. **备选 1**：情绪更强烈
2. **备选 2**：角度不同
3. **备选 3**：更简洁或纯视觉

**要求**：
- 都符合字数限制（中文≤22字，英文≤15词）
- 风格与主旁白一致
- Mode 1：无品牌信息
- Mode 2：可包含品牌名或卖点
- 各有侧重点
- 可包含"无（纯视觉）"选项

## 使用场景建议

### 何时使用 Premium 版本？

**推荐使用**：
- 重要创作项目（艺术短片、品牌大片）
- 需要精细执行的项目
- 多版本A/B测试
- 需要完整制作指南的情况

**Standard 版本足够**：
- 快速创意验证
- 初步方向探索
- 小规模测试
- 预算/时间有限

## 质量检查清单（Premium）

除了 Standard 版本的检查项，Premium 还需确认：

- [ ] 字幕 覆盖所有关键时间点（0-2s/2-5s/5-9s/9-12s/12-15s）
- [ ] 字幕 样式与整体风格匹配
- [ ] 音效 在Drop点有强化音效
- [ ] 音效 有开场音和收尾音
- [ ] 备选旁白 提供3个备选
- [ ] 所有备选台词符合字数限制
- [ ] 备选台词各有侧重点
- [ ] Mode 1：字幕和备选旁白无品牌信息
