**一句话描述**：上传主角图+表达意图 → 自动生成带 ReelOS 品牌气质的15秒视频切镜脚本

## 简介

这是一个专业的 **ReelOS 15秒品牌创意视频脚本生成工具**。上传一张主角图片，说出你的创意意图，即可生成完整的15秒视频切镜脚本，包含画面描述、旁白文案和BGM建议。

支持**纯创意模式**（艺术/哲学/情感/文化表达）和**品牌广告模式**（产品推广/营销），默认强化 ReelOS 式高级、清爽、强记忆点的品牌表达，适合短视频创作、艺术表达、品牌营销等场景。

## 使用指南

### 输入参数

#### 必填
- **主角图片**：1张清晰图片（或详细描述）
- **表达意图**：想要的感觉/风格/主题

#### 可选
- **语气风格**：可爱/高级/酷飒/史诗/俏皮/硬核/空灵/动感/极简
- **节日场景**：春节/新年/情人节/黑五/毕业季等
- **节奏偏好**：动感快切/舒缓推进/史诗爆发/极简留白
- **画面比例**：9:16（竖屏）/ 16:9（横屏），默认 9:16
- **必须旁白**：指定台词内容
- **期望BPM**：音乐节奏

#### Mode 2 专属（提供产品信息时自动启用）
- **产品/品牌信息**：产品名/核心卖点/品牌理念
- **植入要求**：自然融入/强曝光/隐喻表达
- **CTA**：立即试用/点击生成/了解更多

### 6种风格模板

#### 1. 节日氛围（FESTIVAL_VIBES）
- **元素**：节日符号/文化元素/氛围渲染/情绪共鸣
- **节奏**：快切 + 节日元素密集
- **BGM**：国潮电子/民族融合 110-120 BPM
- **适合**：春节/圣诞/情人节等节日创意、文化表达

#### 2. 时尚视觉（FASHION_VISUAL）
- **元素**：闪光灯/霓虹/HUD线条/材质特写
- **节奏**：卡点切镜 + 强节拍
- **BGM**：House/Trap 120-130 BPM
- **适合**：时尚潮流、街头文化、年轻受众

#### 3. 史诗叙事（EPIC_NARRATIVE）
- **元素**：能量特效/冲击波/粒子爆发/慢动作
- **节奏**：慢动作 + 爆发点
- **BGM**：史诗电子/交响乐 130-150 BPM
- **适合**：奇幻、武侠、游戏IP、视觉震撼

#### 4. 极简艺术（MINIMAL_ART）
- **元素**：留白/材质质感/微动效/几何线条
- **节奏**：舒缓推进 + 精准卡点
- **BGM**：Ambient/Minimal 90-110 BPM
- **适合**：哲学思考、禅意表达、高级感、艺术短片

#### 5. 情感共鸣（EMOTIONAL_CONNECT）
- **元素**：人物特写/情绪渐进/故事感/回忆氛围
- **节奏**：情绪曲线递进
- **BGM**：钢琴/吉他/情感电子 100-120 BPM
- **适合**：人文故事、情感表达、回忆杀、温情主题

#### 6. 品牌广告（BRAND_AD）（仅 Mode 2）
- **元素**：产品展示 + 创意隐喻 + 信息植入
- **节奏**：信息密集 + 清晰结构
- **BGM**：节奏明确 110-130 BPM
- **适合**：产品推广、品牌宣传、营销活动

## 常见问题

### Q: 如何判断 Mode 1 还是 Mode 2？
A: 未提供产品/品牌信息 → Mode 1（纯创意）；明确提供产品信息 → Mode 2（品牌广告）。

### Q: 不上传图片可以吗？
A: 可以。根据意图生成通用脚本，但上传图片能获得更精准的定制效果。

### Q: 可以混合多种风格吗？
A: 可以！"春节+时尚"、"仙侠+科技"，智能混搭创造独特视觉体验。

### Q: 旁白超过22字怎么办？
A: 拆分成两句分段说，或改为纯视觉表达（文字动效），或标注"无（纯视觉）"。

### Q: 如何让产品植入更自然？
A: 使用视觉隐喻而非硬广。在BGM的Drop点展示关键信息，让产品融入故事而非打断故事。

### Q: 如何选择合适的旁白声音？
A: 参考 `reference/voice_design.md`，包含10种声音类型和场景匹配矩阵。
