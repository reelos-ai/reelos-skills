# 时尚动感案例库

本案例库展示 Mode 1（纯创意）与 Mode 2（品牌广告）在时尚场景中的应用差异。

---

## Mode 1：纯创意（无品牌/无产品植入）

### 案例 1：街头自由 - 态度宣言

#### 输入参数
```yaml
主角图片: 街头滑板女生，动作定格，潮流混搭
创意意图: 街头自由 + 态度表达 + 年轻宣言
语气风格: 叛逆、自由、真实
画面比例: 9:16
模式: Mode 1（纯创意）
```

#### 输出脚本
```
画面：0-2s定格跃起滑板悬空墙面涂鸦线条爆发，2-5s快切落地冲击波街头元素如碎片飞溅耳机链条衣角飘逸，5-9s推镜侧脸特写眼神坚定城市霓虹在瞳孔中流转自由二字如光影浮现，9-12s环绕360度脚下裂纹扩散涂鸦演化成态度宣言墙，12-15s定镜回眸笑容画面定格黑白闪回留下态度不留遗憾。

旁白：自由不问方向，态度才是终点。

音乐：Trap 节奏 125BPM，11s Drop 带破音效果。
```

#### 设计要点
- **纯表达主题**：街头文化、自由精神、态度宣言
- **视觉隐喻**：涂鸦线条=自由表达，裂纹扩散=影响力扩散
- **色彩**：黑白+霓虹紫+橙红涂鸦
- **节奏**：强卡点，定格+冲击波+环绕
- **无品牌元素**：纯粹艺术表达，无商业内容

#### 输入参数
```yaml
主角图片: 黑色皮衣街头女生，侧脸凝视
创意意图: 时尚动感 + 酷飒 + 年轻受众

```

- **色彩**：黑+紫霓虹+蓝+粉
- **节奏**：强卡点，定格+快切结合

## 通用设计模式

### Mode 1 vs Mode 2 对比

| 维度 | Mode 1（纯创意） | Mode 2（品牌广告） |
|------|------------------|-------------------|
| **核心诉求** | 艺术/情感/文化表达 | 产品功能/品牌价值传递 |
| **视觉重点** | 纯视觉隐喻，哲学意境 | 产品功能视觉化，自然植入 |
| **旁白风格** | 诗意、哲理、情感共鸣 | 可包含品牌名/核心价值主张 |
| **时间分配** | 0-3s冲击，3-8s展开，8-12s高潮，12-15s升华 | 0-5s吸引，5-9s产品首次触达，9-12s功能深化，12-15s品牌强化+CTA |
| **CTA策略** | 无CTA，留白回味 | 12-15s必含Logo+CTA |
| **适用场景** | 艺术片/情感表达/文化输出/节日氛围 | 产品推广/品牌campaign/营销活动 |

### 产品植入时机（Mode 2 时尚场景）

- **2-5s**：灯牌/丝带/DJ台 → 流程展示（脚本→成片）
- **5-9s**：四个旋钮/篮板灯/相册页 → 四类型视频
- **9-12s**：光道/镜面/时光机 → AI Router/导演引擎
- **12-15s**：Logo + 品牌旁白 + CTA

**Mode 1 无产品植入**：所有时间段聚焦艺术表达和情感共鸣。

### 旁白设计差异

**Mode 1（纯创意）旁白特征**：
- 诗意哲理，情感共鸣
- 无品牌名，无商业诉求
- 例："自由不问方向，态度才是终点"
- 例："每一帧光影，都是时光的情书"

**Mode 2（品牌广告）旁白特征**：
- 可包含品牌名（AI 导演）
- 核心价值主张清晰
- 例："AI 导演，让创意不设限"
- 例："创意如 T 台，AI 导演就是你的聚光灯"

### 视觉隐喻策略

**Mode 1**：
- 纯艺术隐喻：涂鸦线条=自由表达、光影=时光质感
- 无产品关联，聚焦情感/哲学/文化

**Mode 2**：
- 产品功能视觉化：灯牌点亮=流程串联、光道分流=AI Router
- 自然融入，避免硬广

### 节奏密度对比

| 类型 | 切镜频率 | 视觉密度 | 信息密度 |
|------|----------|----------|----------|
| Mode 1 | 中等（留白+冲击） | 中高（艺术化） | 低（纯情感） |
| Mode 2 | 高（信息传递） | 高（产品展示） | 高（功能+品牌） |
