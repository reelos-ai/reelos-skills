## 模式 1：纯创意案例（无品牌植入）

### 案例 1：春节团圆 - 温暖回家

#### 输入参数
```yaml
主角图片: 年轻人拉着行李箱，温暖笑容
创意意图: 春节回家团圆 + 温暖治愈 + 游子归乡
语气风格: 温暖、治愈、情感
画面比例: 9:16
```

#### 输出脚本
```
画面：0-2s特写行李箱轮子滚动车站灯光流转福字倒影，2-5s快切买票进站检票窗花纹样随人流飘动像思念拼成回家路，5-9s推镜车窗外城市灯火向后退去红灯笼一串串闪过像时光倒流回童年，9-12s拉远火车驶入小镇雪夜万家灯火亮起每扇窗都是温暖的故事，12-15s定镜推开家门灯光溢出全家人笑容定格烟花在窗外绽放。

旁白：最远的路，是回家最近的心。

音乐：温馨民谣风95BPM，9s弦乐渐强Drop，尾段钢琴+提琴收尾。
```

#### 设计要点
- **情感主题**：游子归乡、温暖团圆、家的呼唤
- **视觉隐喻**：窗花飘动=思念、灯笼闪过=时光倒流、万家灯火=归属感
- **色彩**：暖黄+橙红+雪夜蓝，温暖对比
- **节奏**：舒缓渐进，9-12s情感高潮
- **纯创意特征**：无品牌、无产品、纯情感共鸣

## 通用设计模式

### 模式 1 vs 模式 2 设计差异

| 维度 | 模式 1（纯创意） | 模式 2（品牌广告） |
|------|-----------------|-------------------|
| **核心目标** | 艺术表达、情感共鸣、文化传承 | 产品信息传达 + 品牌价值传递 |
| **旁白风格** | 诗意、哲理、情感化（≤22字） | 可含品牌名/核心卖点（≤22字） |
| **5-9s段落** | 视觉隐喻深化、情感推进 | 首次产品触点（视觉隐喻植入） |
| **9-12s段落** | 情感/哲理高潮、意境升华 | 产品特性/品牌理念深化 |
| **12-15s段落** | 余韵、回味、lasting impression | 品牌强化 + Logo + CTA |
| **视觉语言** | 纯粹艺术化、象征性 | 自然融合产品元素 |
| **信息密度** | 低（留白多、意境足） | 中高（信息点清晰但不硬广） |

### 产品植入时机（模式 2 专属）

- **5-7s**：窗花/春联展开→一站式流程
- **7-9s**：四盏灯笼/四扇门→四类型视频
- **9-11s**：红绸选路/烟花分流→AI Router
- **11-13s**：元宝纹样/祥云图案→品牌价值
- **13-15s**：Logo+台词+CTA

**植入原则**：
- 通过视觉隐喻自然融入，不打断叙事
- 产品信息与节日元素结合（如：红包=惊喜功能、灯笼=多类型）
- 避免硬广式直接展示

### 色彩黄金比例

**模式 1（纯创意）**：
- **主题色**：50%（根据情感调性选择：暖黄/橙红/蓝紫）
- **节日红**：30%（点缀性使用）
- **金色/白色**：15%（强调/留白）
- **环境色**：5%（氛围营造）

**模式 2（品牌广告）**：
- **中国红**：50%（主导色）
- **金色**：30%（强调色）
- **暖黄/橙**：15%（辅助色）
- **品牌色**：5%（点缀色，与节日色融合）

### 旁白设计原则

**模式 1（纯创意）**：
- 诗意化表达（如："一盏灯，照亮千年的等待"）
- 哲理性思考（如："最远的路，是回家最近的心"）
- 情感共鸣点（触动人心、引发思考）
- 避免任何品牌/产品信息

**模式 2（品牌广告）**：
- 可含品牌名（如："AI 导演，让国潮创意全球开花"）
- 核心卖点浓缩（如："把灵感，拍成好彩头"）
- 品牌价值主张（如："每一帧，都是家的温度"）
- 节奏朗朗上口，易记易传播

### 视觉隐喻库（春节主题）

**模式 1 专属隐喻**（纯艺术表达）：
- 窗花飘动 = 思念流转
- 灯笼闪过 = 时光倒流
- 星河倒挂 = 梦境意境
- 天上人间 = 双重世界
- 万家灯火 = 归属感

**模式 2 专属隐喻**（产品功能可视化）：
- 窗花贴满 = 流程展开
- 红绸选路 = AI Router智能路由
- 元宝纹样 = 财富价值
- 红包打开 = 功能惊喜
- 财神爷 = 价值传递
- 玉玺盖印 = 品牌认证
- 光纤导向 = 技术优势

**通用隐喻**（两种模式皆可用）：
- 烟花绽放 = 高潮/惊喜
- 春联点亮 = 对称美/仪式感
- 福字浮现 = 祝福/开端
- 舞狮跳跃 = 欢庆/活力

### 时间结构设计

**模式 1（纯创意）情感弧线**：
```
0-3s:  视觉冲击，建立主题基调
3-8s:  核心概念展开，视觉隐喻深化
8-12s: 情感/哲理高潮，意境升华
12-15s: 余韵回味，lasting impression
```

**模式 2（品牌广告）信息弧线**：
```
0-5s:  Hook - 吸引注意力的视觉冲击
5-9s:  首次产品触点（通过视觉隐喻）
9-12s: 产品特性/品牌理念深化
12-15s: 品牌强化 + 旁白 + Logo + CTA
```

### 镜头语言节奏

**模式 1**：
- 舒缓流畅为主（慢镜、推镜、环绕）
- 快切用于情绪转折点
- 留白镜头营造意境（定镜、微动）
- 平均每段3-5秒（较模式2更长）

**模式 2**：
- 密集快切（每2-3秒一个视觉变化）
- 强节奏卡点
- 信息点清晰可见
- 产品露出时适当放慢（让观众看清）
